# 🏛️ dao-treasury-manager

DAO treasury management with multi-sig, streaming and budgeting

## Features

- Multi-sig treasury
- Payment streaming
- Budget allocation
- Expense tracking
- Reporting

## Tech Stack

- **Smart Contracts:** Solidity ^0.8.20
- **Framework:** Hardhat
- **Frontend:** Next.js + TypeScript
- **Web3:** ethers.js / wagmi

## Getting Started

```bash
npm install
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.js --network localhost
```
